Essential Java Built-in Classes

Input/Output Classes

Class Description Common Usage
Scanner Used for reading input from various sources (keyboard, files, etc.) Scanner scanner = new Scanner(System.in);
String input = scanner.nextLine();
System Provides standard input/output streams and utility methods System.out.println("Hello");
System.err.println("Error");

Wrapper Classes

Class Description Common Usage
Integer Wraps primitive int type, used in collections Integer num = Integer.valueOf(42);
int x = Integer.parseInt("42");
Double Wraps primitive double type, used in collections Double pi = Double.valueOf(3.14);
double d = Double.parseDouble("3.14");
Boolean Wraps primitive boolean type, used in collections Boolean flag = Boolean.valueOf(true);
boolean b = Boolean.parseBoolean("true");

Note: Wrapper classes are essential when using collections (ArrayList, HashSet, etc.) as they don't accept primitive types.

String Handling

Class Description Common Usage
String Represents text as a sequence of characters String str = "Hello";
str = str.toUpperCase();
StringBuilder Mutable sequence of characters, efficient for string manipulation StringBuilder sb = new StringBuilder();
sb.append("Hello").append(" World");

Collections

Class Description Common Usage
ArrayList Resizable array implementation of the List interface ArrayList<String> list = new ArrayList<>();
list.add("item");
HashMap Implementation of Map interface using hash table HashMap<String, Integer> map = new HashMap<>();
map.put("key", 42);

Note: Collections are fundamental for storing and manipulating groups of objects.

Math and Numbers

Class Description Common Usage
Math Provides basic math operations and constants double sqrt = Math.sqrt(16);
double random = Math.random();
Random Generates pseudo-random numbers Random rand = new Random();
int num = rand.nextInt(100);
